## Requirements

- Проект должен запускаться локально.
- Должна быть инструкция запуска.
- Должен быть рабочий `/start`.
- Должна быть регистрация пользователя.
- Должна быть привязка пользователя к дому.
- Должен быть сценарий создания заказа на вынос мусора.
- Исполнители должны получать заказы только своего дома.
- Заказ должен брать только один исполнитель.
- Фото до и после обязательны для trash removal.

Сделай рабочее MVP, которое потом можно улучшать.
